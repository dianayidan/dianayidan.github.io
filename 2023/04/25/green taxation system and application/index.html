Green taxation system

21.1绿色税收助力绿色发展浅析

绿色税收概念及提出背景

  • 税收是政府对经济宏观调控的重要手段之一；绿色税收：投资于防治污染或环境保护的纳税人给予的税收减免

  • 2005:《加快推进生态文明建设的意见》：明确提出绿色化的概念，环境保护费改税制度开启

  • 2016:《中华人民共和国环境保护税法》：绿色税收体系开始新的时期

我国现行的绿色税收制度

  • 环境保护税

    • 事后干预型绿色税收：2016:《中华人民共和国环境保护税法》,2018实施；首部专门为环境保护设立的独立型绿色税种（我国唯一独立的绿色税种）

  • 现行税制中其他具有环保性质的绿色相关税种 (其他绿色相关税种)

    • 资源占用型:提高资源利用率，促进国家资源合理开发利用，减少资源损失浪费

      • 资源税、城镇土地使用税、耕地占用税

    • 行为引导型:设置部分与环境保护相关的税目、税率差异和税收优惠政策对污染环境的特定消费品的生产、消费或使用行为征税(增加特定消费者税收负担抑制对高污染高耗能产品的消费)

      • 消费税、车船税、车辆购置税

  • 与环境保护相关的税收优惠政策

    • 增值税：

    • 企业所得税：资源综合利用产品和服务、节能服务公司等方面制定了相关税收优惠政策（促进企业节约利用资源，增强环境保护意识，鼓励环保产业发展）

绿色税收助推绿色发展

  • 绿色税收刺激企业进行绿色技术创新

    • 国家实施规模减税降费政策；1.为企业提供高新技术企业研发费用加计扣除辅导2.降低经营成本3.减免税额用于高端技术人才的招录，加大科技创新资金投入

  • 绿色税收能推动绿色低碳发展进程

    • 环境保护税、资源税等绿色税制充分发挥税收杠杆的绿色调节作用；1.对重点污染物的减排目标2.促进经济结构优化和发展方式转变3.助力国家低碳减排路线

我国绿色税收面临的问题

  • 绿色税收缺乏完整的税收体系

    • 绿色税收占总税收比重小

    • 税率标准偏低，纳税人在治污减排耗费的成本高

    • 绿色税务协作：分工不清晰环保信息衔接不够通畅

  • 绿色税收理念渗透不足

    • 企业：以自身利益为首要条件忽略绿色发展

    • 民众：绿色税收含义模糊，污染物排放及耗能意识不足

我国绿色税收发展相关建议

  • 改善绿色税收管理体系

    • 扩大绿色税收征收范围

      • 资源税的征收范围不断扩大，政府适当提高税率

    • 完善绿色税收计税依据

      • 调整部分税的计税依据使其更合理

    • 改善各部门间协同

      • 改善各部门与环保部门的合作

    • 推进信息共享:发挥数据平台作用

  • 加强绿色税收对企业的激励机制

    • 减税降费

      • 降低企业成本，加大低碳环保投入力度

    • 企业所得税对环境保护及碳减排的支持力度

      • 环保产品和技术上给予税收优惠支持，尤其在碳减排项目与设备采购方面

    • 完善多排多缴、少排少缴、不排不缴的正向激励机制

      • 激发企业绿色低碳发展，引导企业减少污染物的排放

  • 推广绿色税收的宣传引导效应

    • 税务部门

      • 落实各项绿色税收政策和优惠（加强对基层税务人员进行培训工作）

    • 纳税人

      • 多种途径进行环保税收政策宣传；使纳税人在执行环保税中得到实惠

      • 展示绿色税收带来的效益，引导纳税人从被动到主动的环保意识

21.2绿色金融助力绿色建筑发展

我国绿色建筑发展现状

  • 建筑面积稳步增长，发展规模不断扩大

  • 政策框架基本建立，标准体系逐步清晰

  • 绿色技术不断发展，建筑成本逐渐下降

    • 增加了绿色建筑的普及率&市场接受率

  • 政府主导推动发展，市场机制尚未形成

    • 需求方面：房屋购买是居民理财的重要途径之一，会重点考虑地段、配套资源很少考虑房屋的绿色低碳性能

    • 供给方面：精细的设计和复杂的操作；融资难和期限错配无法从绿色建筑的正外部性获取经济利益

    • 短期：政府主导；长期：市场化机制

绿色金融助力绿色建筑发展的路径

  • 绿色金融缓解绿色建筑资金短缺促进行业高质量发展

    • 绿色认证：降低了项目的准入风险

    • 风险分摊机制：鼓励资金流入绿色建筑领域

  • 绿色金融解决绿色建筑期限错配问题

    • 建筑设计与运行之间的性能错配

    • 绿色信贷发放与绿色性能认定的时间错配问题

    • 绿色保险能解决两大错配问题

      • 开发商通过在项目建造前购买绿色建筑保险确保建筑运行阶段的绿色节能性能；银行通过保险单为开发商发放绿色信贷项目完成后对实际建筑进行绿色节能性评价，若未实现将由保险公司按照合同进行赔付

  • 绿色金融促进绿色建筑指标体系建立及信息披露

    • 绿色金融标准：开发商使用绿色金融产品的前提是通过绿色建筑项目认证；绿色建筑标准转化为绿色金融标准

      -

    • 绿色建筑的低碳效益量化评估体系：金融产品的定价；绿色建筑的绿色低碳效益进行量化；绿色建筑的低碳效益量化评估体系的建立

案例分析：浙江湖州创建绿色建筑和绿色金融协调发展试点城市

  • 政策引领激发绿色建筑产业活力

  • 标准推动绿色建筑和绿色金融的有效衔接

  • 产品创新完善绿色建筑金融服务体系

  • 科技支撑打造绿色建筑投资数字化平台

绿色金融支持绿色建筑发展建议

  • 政策引导促进绿色建筑与绿色金融协同发展

    • 绿色建筑与绿色金融协同发展

    • 绿色金额支持绿色建筑的政策环境

    • 未形成一套独完善的实践指导政策体系

  • 完善认证体系，提高绿色金融对绿色建筑项目的识别和管理效率

  • 创新绿色金融产品种类适用绿色建筑发展多重需要

  • 推广绿色金融与绿色建筑的协同发展理念

22.1 年金基金管理机构的ESG投资实践分析

我国养老金“三支柱”模式：（三支柱养老保障体系）

  • 基本养老保险为基础，以企业和职业年金为补充，与个人储蓄型养老保险和商业养老保险相连接

  • 第一支柱占主导地位，第二支柱惠及人口有限，覆盖率不高，第三支柱仍处于探索阶段

  • 我国养老金体系呈现第一支柱独大，第二三支柱养老金积累有限的结构性失衡问题

职业年金与企业年金的发展现状

  • 养老金第二支柱：企事业单位发起，商业机构运作的“职业养老金”
    • 企业年金：面向企业职工，职业年金：面向机关事业单位
    • 职业年金：公务员法管理的单位，具备强制性，发展速度有限

年基金管理机构践行ESG投资的必要性

  • 长期性：缴费确定型的完全积累制，资产管理人的受托责任
  • 公共属性：受益人是社会公众
  • 避险属性：投资时避免对社会和环境产生重大的负面影响（ESG投资的负面筛选exclusionary screening 可以有效规避）
  • 追求稳定收益：ESG投资侧重绿色产业投资，资金流向生态和谐领域，可以有效应对气候变化带来的系统性风险

海外年基金管理机构的ESG投资实践案例（英国大学养老金计划USS）

  • ESG整合：ESG因子、财务因子、投资组合决策、环境和社会投资价值
  • 公司参与和股东行为：负责人投资机构、利用股东权利影响公司行为、直接参与或投票方式介入投资项目（避免投资价值受损）
  • 风险管理：
    • 第一道防线：业务部门（风险归类和控制）
    • 第二道防线：风险部门（风险监督）
    • 第三道防线：内部审计部门（独立审查）

我国年基金管理机构践行ESG投资的建议

  • 具备坚定的ESG投资理念与目标：关注ESG细分风险，环境绩效管理，腐败风险

  • 基于自身优势明确ESG投资的管理组织架构：

    • 董事会：全面监管ESG事宜
    • 管理层：制定ESG投资计划和考核政策
    • 业务层：执行落实并定期汇报的一体化ESG管理架构

  • 构建ESG投资机制，参与被投公司治理

    • 制度层面：构建适合的ESG投资机制，制定高碳投资排除机制以减少高碳领域投资对冲气候变化带来的转型风险
    • 实践层面：
      • 投资之前：构建ESG因子评估框架体系，对投资标的全面评估
      • 投资过程：对投资标的ESG风险进行持续监控和管理
      • 投资管理：发挥股东积极性，参与公司治理，阻挡公司ESG风险蔓延，规避相关投资风险

  • 创新ESG养老产品，推进行业各方合作

    • ESG养老产品发行和ESG主题投资：
        1. 规避风险：投资绿色资产支持证券（ABS）、环境友好型债券计划产品
        2. 借鉴国外经验：形成具有中长期可回溯产品业绩
    • 行业各方协同推出年基金的ESG投资进程：
      • 在评估体系、产品设计、风险防控方面共同助力年基金践行ESG理念

22.2金融科技如何推动绿色信贷发展

金融科技在绿色信贷领域的应用

  • 绿色信贷业务营销

    • 针对性营销业务：寻找客户和项目具有环境正外部性 （通过大数据，云计算）
    • 匹配：将这些客户的留存信息与当前绿色信贷政策匹配
    • 筛选：符合条件的客户目标，根据匹配的结果，商业银行可以提高业务营销的成功率
    • 绿色信贷服务平台：
      • 提供评估认证、政策查询
      • 产品咨询
      • 信贷发放

  • 绿色信贷智慧定价（复杂）

    • 量化分析绿色信贷中的环境效益：定制算法
    • 对贷款利率水平进行调整：根据客户经营情况变化，实现绿色信贷定价的精准化、动态化和差异化
    • 提高绿色信贷对于不同客户的适用性，降低客户融资成本

  • 绿色信贷风险管理——金融科技的参与

  • 通过多种方式降低商业银行面临的环境风险

  • 数据检测和收集

  • 降低“洗绿”和“漂绿”等行为风险

金融科技推动绿色信贷发展面临的挑战

  • 增加监管难度：去中心化和金融脱媒，绿色信贷当前还在发展阶段
  • 公开环境数据质量不高：
    • 环境数据的有效性和准确性影响了环境风险建模分析的效果 （但数据不可溯源影响数据质量）：公共数据更新不及时，溯源难，数据质量低
  • 金融科技在绿色信贷业务的实用度低：
    • 绿色信贷发展时间不长：缺乏经验和相关数据支持无法展开相关数据分析
    • 金融科技在绿色信贷业务中的实用度低：金融科技成熟度不高；绿色与非绿色项目划分模糊

相关政策建议和展望

  • 支持绿色金融科技监管沙盒
  • 鼓励发展与绿色信贷更契合的金融科技技术
    • 金融科技方法学：
      • 绿色行业关键字抓取
      • 绿色行业智能识别与分类
      • 特定环境影响因子分析
      • 提高模型敏感度和准确度
  • 建立环境公开数据统计共享平台，鼓励市场主体开展信息披露

23.1 公共建筑能效提升的国内外实践分析

公共建筑能效

  • 定义：公共建筑在使用阶段的运营能耗（涵盖建筑材料的生产，建筑施工、使用、拆除等）
  • 能效提升路径：新型墙体材料，建筑用能设备效率、运行效率，合理设计建筑围护结构的热工性能

公共建筑能效提升的国内实践

  • 政策层面：
    1. 法律法规：《建筑节能管理条例》《民用建筑节能条例》
    2. 规范标准：《绿色建筑评份标准》
    3. 规划行动《建筑节能与绿色建筑“十三五”规划》
  • 技术层面：根据《公共建筑节能设计标准》，我国公共建筑主要围绕围护结构、供热制冷、排水、电气等，以提升运行效率、运用高效用能设备以及调整建筑结构三种路径实现建筑更新改造与能效提升
    1. 运行效率
    2. 高效用能设备
    3. 调整建筑结构
  • 市场层面：节能服务公司通过与客户签订节能服务合同及由合同能源管理模式衍生的其他方式为公共建筑能效提升项目提供节能服务，并从项目进行节能改造后获得的节能效益中收回投资和取得利润
    1. 节能效益分享型（50%）
    2. 节能量保证型（40%）
    3. 设备租赁型
    4. 能源费用托管型
